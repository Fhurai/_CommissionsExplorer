Introduction
	Une application web permettant d'accéder aux commission d'un répertoire pour les consulter, que la commission soit image, vidéo ou musique.
	
Fonctionnalités
	Liste des artistes dans le répertoire avec un thumbnail pour le dossier. Si aucun thumbnail n'existe pour l'artiste alors affichage d'une image par défaut.
	Cliquer sur le dossier d'un artiste ouvre ce dossier (Transition avance) listant toutes les commissions faites par l'artiste.
	Cliquer sur une commission ouvre une modale affichant la commission choisie avec bouton d'ouverture dans un nouvel onglet et bouton de téléchargement.
	Possibilité de visionner statistiques de commissions (nb commissions, nb artistes)
	Administration:
		+ Génération des fichiers json pour sfw et twitter.
		+ Génération du jeton nsfw.
		+ Si nombre de dossiers pictures est différent de nombre de dossiers thumbs, générer les fichiers json.
	Stats
		+ Nombre de dossiers pictures SFW/NSFW.
		+ Nombre de dossiers thumbs SFW/NSFW.
		+ Nombre de dessins SFW/NSFW
	Différence entre nombre dossiers pictures NSFW et Json sfw twitter => Génération JSONs.


Webservices
	+ Génération d'un jeton pour activer la partie nsfw.
	+ Génération fichiers json (sfw & twitter)
		-> Utilisation par défaut si un des deux fichiers est manquant.
	+ Liste de tous les artistes.
		-> Liste avec paramètre contenu, lien twitter, lien premier thumbnail (image vierge si aucun thumbnail existant).
		-> Prévoir un paramètre à l'appel pour ne récupérer que les artistes SFW.
	+ Liste des artworks d'un artiste.
		-> Liste avec lien thumbnail, lien artwork HQ, type d'artwork (dessin, vidéo, musique, texte).
		-> Affichage des artworks dans une gallerie dédiée à l'artiste.
		-> Cliquer sur un artwork affiche une modale d'affichage de l'artwork dans une fenêtre qui fait maximum 80% de la hauteur ou 80% de la largeur.
